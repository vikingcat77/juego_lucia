# Gatitos y Cajitas

Juego web tipo `Block Puzzle` / `TenTrix`.

## Base actual

- Tablero `8x8`.
- 3 piezas disponibles por turno.
- Arrastre con vista previa.
- Limpieza de filas y columnas completas.
- Puntuacion y mejor marca guardada en `localStorage`.

## Archivos

- `index.html`: estructura de la interfaz.
- `style.css`: estilos y disposicion responsive.
- `script.js`: logica del tablero, piezas y puntuacion.
